# Weather Diary - Дневник погоды

## Информация об авторе
**Автор:** Александр Томшин


## Описание программы
Weather Diary - это графическое приложение для ведения дневника погоды. Программа позволяет:
- Добавлять записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в удобной таблице
- Фильтровать записи по дате и температуре
- Сохранять данные в JSON файл и загружать из него
- Удалять отдельные записи или все сразу

## Требования к системе
- Python 3.7 или выше
- Операционная система: Windows, macOS, Linux
